# Hexa Shoot Arena

六角形の壁付きコートで戦う、スマホ向け2対2アーケードシュートゲームです。

このリポジトリは、旧試作版を継ぎ足さずに作り直した **新規プロジェクト v0.1** です。

## v0.1で実装済み

- Phaser 3 + TypeScript + Vite
- タイトル／試合設定画面
- 5チーム
- 六角形コート
- ゴール横の内向きカーブ壁
- 8方向移動
- プレイヤー／キーパー切り替え
- ゴロパス
- ストレートシュート
- 壁反射
- ボール摩擦と停止
- 人に当たったボールが横へ弱くこぼれる
- 未操作キーパーのゴール中央帰還
- 簡易CPU
- 1分／2分試合
- スマホ用仮想スティックとABCD表示

## 起動方法

Node.js 20以降を推奨します。

```bash
npm install
npm run dev
```

本番ビルド：

```bash
npm run build
```

`dist/` の内容をGitHub Pagesへ公開できます。

## PC操作

- `WASD`: 8方向移動
- `J`: パス
- `K`: シュート
- `L`: 操作選手切り替え
