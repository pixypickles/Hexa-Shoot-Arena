# Hexa Shoot Arena — HTML版 v0.1

GitHub Pagesへそのままアップロードできる、依存関係なしのHTML/CSS/JavaScript版です。

## ファイル

- `index.html`
- `style.css`
- `game.js`
- `SPEC.md`

## 公開方法

1. このフォルダの中身をGitHubリポジトリ直下へアップロード
2. GitHubの `Settings`
3. `Pages`
4. `Deploy from a branch`
5. Branchを `main`、Folderを `/ (root)` に設定
6. `Save`

npm、Node.js、Viteは不要です。

## 現在の実装

- 5チーム
- 試合時間1分／2分
- 六角形コート
- ゴール横の内向きカーブ壁
- 8方向移動
- ゴロパス／浮き玉パス
- ストレート／回転シュート
- 壁反射
- ボールの自然減速
- 人に当たると横へ弱くこぼれる
- 弱い球のトラップ
- キーパー自動帰還
- 簡易CPU
- 6秒ルール
- 浮き球への自動ボレー

## v0.2 シュート完成版

- C長押しでストレートチャージ
- D長押しで回転チャージ
- 0.18秒以上でハーフ、0.48秒以上でフルチャージ
- C/Dの素早い2回押しでハーフチャージ
- フルチャージのストレートはブレ球
- チャージゲージを画面下部へ表示
- 回転量を弱め、壁沿いに吸い付きにくく調整
- 選手への強い接触後、0.3秒間は再接触しない
- 強い球は正面へ返さず、横へ大きくこぼれて速度を18%まで失う
- 低速になった球はトラップを優先
- 選手同士の間でボールが弾き合い続ける問題を軽減
