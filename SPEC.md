# Hexa Shoot Arena 正式仕様

## 固定事項

- 六角形コート
- ゴール横は内向きカーブ壁
- 全周壁あり、ボールアウトなし
- 1チームはキーパー1名、フィールド1名
- 選手はセンターラインを越えない
- 全得点は1点
- 試合時間は1分または2分
- チーム間の能力差なし
- 左は8方向入力、右はABCDボタン
- ボールは通常より約15%大きく表示
- 転がるボールは自然に減速して止まる
- 人に当たった強いシュートは速度を大幅に失い、正面でなく横へこぼれる
- 勢いの弱いボールは人に付いて止まる
- 未操作キーパーはゴール中央へ自動帰還
- 相手コートへ6秒以内にボールを移せない場合は相手ボール

## 攻撃操作

- A: 高速ゴロパス。押した瞬間に受け手へ操作交代
- B: ゆっくりした浮き玉パス。押した瞬間に受け手へ操作交代
- C: ストレートシュート。長押しでチャージ、最大でブレ球
- D: 回転シュート。長押しで速度と変化量上昇
- CまたはDを素早く2回: ハーフチャージ
- 必殺ゲージ満タン時の発動操作は実装段階で固定する

## 守備操作

- A: 操作キャラ交代
- B: 撃ち返し
- C: 飛距離長めのパンチングジャンプ
- D: 飛距離短めのキャッチジャンプ
- ジャンプ中または立った状態で、接触タイミングにC/Dを押すとジャストキャッチ
- ブレ球は通常キャッチだけでは失敗する

## 浮き球

- 浮き球はスライディングで防げない
- 受け手は多少遠くても踏み込んでボレーを優先
- 高い球はオーバーヘッドを自動選択

## 必殺シュート

試合前に1種類選択。必殺ゲージはチーム共有。

1. Hook Shot: キーパー前で左右どちらかへ約30度急変化
2. Side Net Curve: 左右どちらかのサイドネットへ大きな弧を描く
3. Break Shot: 遅いがFPを必ず吹き飛ばす。GKもジャストキャッチ以外は吹き飛ぶ
4. Stealth Shot: 0.5秒ボールが消え、床の影だけ表示される

## チーム

1. BLIZZARD FOX: 白黒縦縞、水色差し色、黒パンツ
2. SALVIDA A: エンジ、白パンツ
3. SALVIDA B: ターコイズ、白パンツ
4. TAKE-ZO: ピンクとネイビーの太ボーダー、白パンツ
5. 漫チェスターP: 全身濃いネイビー
